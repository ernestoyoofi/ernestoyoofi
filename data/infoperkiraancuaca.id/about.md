# Tentang Aplikasi

Aplikasi ini adalah aplikasi yang hanya untuk prototipe (bentuk dasar percobaan) pada sistem notifikasi / realtime notification

Aplikasi ini sesungguhnya hanya menkoleksi data dari beberapa data sesuai apa yang perlihatkan dalam aplikasi.
Sumber asli (Original Source) aplikasi ada pada bawah ini :
- Gempabumi : https://data.bmkg.go.id/gempabumi/ (dokumentasi)
- Gempabumi Data : https://data.bmkg.go.id/DataMKG/TEWS/autogempa.json
- Gempabumi List : https://infoperkiraanalam-id.vercel.app/gempabumi/view-all
