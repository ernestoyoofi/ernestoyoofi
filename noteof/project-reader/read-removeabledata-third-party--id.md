## Penghapusan data dari suatu projek

Sebagai developer tau bahwa kamu ingin menghapus data ini demi suatu privasi atau keaman data yang mungkin jadi masalah utama.

Kamu dapat menghapusnya, namun salah satunya cara, kamu coba masuk ke halaman dashboard utama, lalu pilih atau cari opsi hapus akun secara permanen.

Jika opsi hapus akun secara permanen tidak ada, tolong lapor ke pada admin, owner, atau penjaga suatu server untuk menghapus data kamu.

Jika kamu menghapus akun kamu dari suatu project, profile kamu masih dapat dilihat, namun beberapa aksi tidak dapat dilakukan.
