## Cara Mendaftarkan Grup WhatsApp Ke Dalam Database Bot ?

> Bukan bermaksud untuk memaksa kehendak untuk menggunakan bot, namun saya hanya untuk menizinkan saja dalam bentuk sistem donasi agar dapat menyaring sistem perpesanan

1. Buka pada halaman profile donasi [disini](https://trakteer.id/ernestoyoofi?quantity=5) (Range 10K - 30K / 2 Items, dapatkan kartu **"App Supported"** atau **"Supporter"**) untuk memastikannya
2. Pada bagian pesan, kamu masukan seperti di contoh di bawah ini, kamu ganti {history_id} ganti pada id yang dikirim bot pada riwayat id, setelah itu jangan lupa check list "Jadikan pesan private"

   Salin pesan dibawah ini jika anda malas mengetik
   
   ```
   # Sistem Dukungan
   history_id: {history_id}

   # Pesan
   Selamat pagi kak !
   ```
   
   <p align="center"><img src="/img/donation.png" width="300px"/></p>
   
4. Lakukan prabayar
5. Jika sudah, kirim pesan ke bot seperti ini

   <p align="center"><img src="/img/check-donation.png" width="500px"/></p>
   
6. Jika tulisan **"Tersimpan pada database"** tertulis "ya" maka, kamu sudah dapat menggunakan, jika tidak harap laporkan.

Ada pertanyaan, tulis di issues / report ke owner
